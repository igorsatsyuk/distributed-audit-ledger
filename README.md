# Distributed Audit Ledger

Event-sourced audit platform with immutable blockchain anchoring.

## Repository Layout

- `backend/` - Java services (CQRS + event sourcing)
- `blockchain/` - smart contracts and scripts
- `frontend/` - Angular UI
- `deploy/` - local infrastructure and environment configs
- `docs/` - architecture and operational documentation

## Current Stage

Project bootstrap is complete and ready for implementation by GitHub Issues (#1+#).
